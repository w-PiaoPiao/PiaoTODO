# LiteToDo 设计规范

## 应用形态

macOS 菜单栏小工具。点击菜单栏图标后弹出一个固定尺寸的面板，所有操作在面板内完成。

## 界面布局

面板宽度约 360px，高度约 480px。整体从上到下分为四个区域：

```
┌─────────────────────────────┐
│  LiteToDo            [⚙设置]│  标题栏
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 买牛奶...                │ │  输入框
│ └─────────────────────────┘ │
├─────────────────────────────┤
│ [工作] [生活] [娱乐] [其他] │  分类按钮区
│  长按筛选 / 长按管理分类    │
├─────────────────────────────┤
│                             │
│  ○ 完成项目报告             │  未完成列表
│    └─ 最新进度：已发初稿    │
│                             │
│  ○ 预约牙医                 │
│    └─ 最新进度：电话未接    │
│                             │
├─────────────────────────────┤
│  ▼ 已完成（2）              │  已完成折叠区
│  ● 看电影《奥本海默》       │
│  ● 整理桌面                 │
└─────────────────────────────┘
```

### 布局说明

- 分类按钮放在输入框下方、未完成列表上方，避免未来手机版被虚拟键盘遮挡
- 未完成列表占据主体空间
- 已完成区域默认折叠在底部
- 标题栏右侧放置设置入口

## 颜色与视觉风格

- 主背景：浅灰/白色系（#FFFFFF / #F5F5F7）
- 主文字：深灰（#1C1C1E）
- 次要文字：中灰（#8E8E93）
- 分类按钮默认颜色：
  - 工作：蓝色 #3B82F6
  - 生活：绿色 #10B981
  - 娱乐：橙色 #F59E0B
  - 其他：灰色 #6B7280
- 完成态文字：浅灰带删除线
- 进度文字：比主文字更浅的灰色
- 圆角：按钮 8px，面板 12px，卡片 10px
- 阴影：菜单栏面板使用轻微阴影，模拟原生弹窗

## 交互设计

### 添加 todo

1. 用户在输入框输入文字
2. 点击底部任意分类按钮
3. 若输入为空，按钮轻微抖动提示
4. 添加成功后，输入框清空，新 todo 出现在列表顶部

### 筛选单个分类

1. 长按某个分类按钮约 500ms
2. 面板轻微震动/按钮高亮反馈
3. 列表只显示该分类的 todo
4. 顶部分类区显示「只看：工作 [取消筛选]」提示
5. 再次长按同一分类或点击取消，恢复全部显示

### 管理分类

1. 长按分类按钮后弹出的菜单包含：
   - 只看此类
   - 编辑名称/颜色
   - 删除分类
2. 删除时若该分类下有 todo，提示「删除后该分类下的 todo 将移入“其他”」
3. 添加分类通过设置面板或分类区的「+」按钮

### 完成 todo

1. 点击 todo 左侧空心圆圈
2. 圆圈变为打勾状态
3. todo 文字添加删除线并变灰
4. todo 移入底部「已完成」折叠区
5. 已完成区自动展开并显示刚完成的项

### 编辑 todo

1. 单击 todo 文字区域
2. 文字变为可编辑输入框
3. 按 Enter 保存，按 Esc 取消
4. 失去焦点时自动保存

### 删除 todo

- Mac 版：右键点击 todo，选择「删除」
- 手机版（未来）：左滑删除

### 添加进度

1. 长按某个 todo 约 500ms
2. 弹出小输入框
3. 输入进度文字后保存
4. 最新进度显示在 todo 文字下方
5. 点击 todo 进入详情，查看全部历史进度

### 拖拽排序

1. 按住 todo 项的拖拽手柄（右侧或左侧图标）
2. 上下拖动调整位置
3. 松开时保存新顺序
4. 排序仅在当前筛选范围内有效

## 空状态

当没有任何 todo 时，列表中央显示：

> 还没有 todo
> 在上方输入内容，然后选择一个分类添加

使用浅灰色小字，避免界面过于空旷。

## 数据结构

### Category（分类）

```json
{
  "id": "work",
  "name": "工作",
  "color": "#3B82F6",
  "order": 0,
  "createdAt": 1719830400000
}
```

### Todo（待办）

```json
{
  "id": "todo-uuid",
  "content": "完成项目报告",
  "categoryId": "work",
  "completed": false,
  "completedAt": null,
  "createdAt": 1719830400000,
  "updatedAt": 1719830400000,
  "order": 0,
  "progressNotes": [
    {
      "id": "note-uuid",
      "content": "已发初稿",
      "createdAt": 1719830500000
    }
  ]
}
```

### AppData（应用数据）

```json
{
  "version": 1,
  "categories": [...],
  "todos": [...],
  "settings": {
    "showCompleted": false,
    "lastSelectedCategory": null
  }
}
```

## 错误与反馈

- 空输入添加 todo：分类按钮轻微抖动，输入框 placeholder 短暂变红
- 保存失败：右上角显示红色 toast「保存失败，请重试」
- 删除分类：二次确认弹窗
- 数据读取失败：尝试从备份恢复，备份也失败时显示引导用户导出日志
