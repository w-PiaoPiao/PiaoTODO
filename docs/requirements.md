# LiteToDo 产品需求文档

## 项目概述

LiteToDo 是一款轻量级、跨平台待办清单软件。首期开发 macOS 菜单栏版本，后续逐步扩展到 Windows、HarmonyOS、Android、iOS。

核心设计哲学：**通过预设分类替代传统加号按钮**，减少用户创建 todo 时的分类决策步骤，让记录更快、分类更自然。

## 目标用户

- 需要快速记录琐事、工作任务的普通用户
- 不喜欢复杂项目层级，偏好轻量、一键分类的用户
- 希望未来在多台设备上保持 todo 同步的用户（远期）

## 核心功能需求

### 1. 菜单栏小工具（macOS 首期）

- 应用在菜单栏显示一个图标
- 点击图标弹出主面板
- 面板尺寸参考：约 360×480 像素
- 支持快捷键 `Cmd + Shift + T` 打开/关闭面板
- 按 `Esc` 关闭面板

### 2. 预设分类与添加 todo

- 默认分类：工作、生活、娱乐、其他
- 分类显示在主面板输入框下方、未完成列表上方
- 用户在输入框输入内容后，点击某个分类按钮，直接将 todo 归入该分类
- 分类按钮不能被未来手机版的虚拟键盘遮挡，因此统一放在输入框下方

### 3. 列表展示

- 默认首页显示所有未完成的 todo
- 长按某个分类按钮，只显示该分类的 todo
- 再次操作可取消筛选，恢复显示全部
- 列表中的 todo 支持上下拖拽排序
- 按创建时间倒序或按用户拖拽顺序展示

### 4. 基础操作

- 标记完成：点击 todo 左侧圆圈，打勾后移入「已完成」折叠区
- 编辑：单击 todo 文字进入编辑模式
- 删除：支持右键删除或侧滑删除（Mac 版优先右键菜单）
- 已完成区域默认折叠，按完成时间倒序排列
- 已完成区域提供「清空已完成」功能

### 5. 进度记录

- 长按某个 todo 项，弹出输入框添加进度
- 进度以文字形式记录，如「已发初稿」「电话未接」
- 列表中只显示最新一条进度
- 点击 todo 可展开查看全部历史进度
- 所有 todo 和进度都记录时间戳，但不在列表中显示

### 6. 分类管理

- 分类可自定义：添加、改名、改色、排序
- 长按分类按钮弹出管理菜单：只看此类 / 编辑 / 删除
- 至少保留 1 个分类，不能全部删除
- 添加分类入口：设置面板或分类菜单

### 7. 设置面板

- 从标题栏齿轮图标进入设置
- 功能包括：管理分类、导出/备份数据、关于

### 8. 数据持久化

- 关闭 App 后 todo 数据保留
- 数据保存在 Mac 本地 JSON 文件
- 每次状态变更后自动保存
- 本期不做云同步，但保留未来扩展可能

### 9. 视觉风格

- 默认采用浅色极简风格，接近苹果原生备忘录
- 界面简洁、留白充足、圆角卡片
- 分类按钮使用不同颜色区分

## 非功能性需求

- 轻量：安装包体积小、启动快、内存占用低
- 稳定：自动保存，避免数据丢失
- 可扩展：代码结构便于后续迁移到 Windows 和手机平台
- 可维护：代码模块化，便于后续功能迭代

## 优先级

1. P0：菜单栏面板、分类按钮、添加 todo、列表展示、完成/删除/编辑、本地持久化
2. P1：进度记录、已完成列表、拖拽排序
3. P2：分类自定义、设置面板、快捷键
4. P3：视觉动画、数据导出、打包发布
